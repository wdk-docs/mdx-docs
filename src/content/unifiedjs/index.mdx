# 内容作为结构化数据

我们将内容编译为语法树，将语法树编译为内容。
我们还提供数百个包来处理介于两者之间的树木。
你可以在 unified 的集体基础上制作各种有趣的东西。

## 建筑

我们提供构建块：从小型、集中的模块化实用程序到将它们组合起来执行更大任务的插件。还有更多。你可以建立在 unified 、混合和匹配的构建块上，共同创造各种有趣的新事物。

## 学

我们提供接口：用于解析、检查、转换和序列化内容。您处理结构化数据。学习如何将构建块插在一起，编写自己的构建块，并用 unified 的方法制作东西。

## Explore

unified 集体今天提供的不断增长的生态系统由 359 个开源项目组成，在 GitHub 上总共有 5.8 万颗星。相比之下，这个集体维护的代码大约是 194 个 Moby Dicks 或 77 个 Linux。在过去的 30 天里，这些项目中维护的 538 个包从 npm 下载了 1b 次。其中大部分由我们的团队维护，其他则由社区提供。
