---
title: 语法树
---

**`syntax-tree`**是一个包含 100 多个项目的组织，这些项目处理基于**[unist][]**的语法树。
这些树通常处理内容: markdown (**[mdast][]**), HTML (**[hast][]**), 自然语言 (**[nlcst][]**), XML (**[xast][]**), 还有 JavaScript (**[esast][]**).

- 要学习 unist，请参阅[它的自述文件][unist]
- 通过使用 **[unified][]** 及其插件，您可以使用语法树而不必处理它们

#### unified

**`syntax-tree`** 是 unified 集合的一部分，该集合将以结构化数据的形式处理内容的组织聚集在一起。

- 有关我们的更多信息，请访问[`unifiedjs.com`][site]
- 关于集体的治理方式，请参见[`unifiedjs/collective`][collective]
- 有关更新，请参阅推特上的[@unifiedjs][twitter]
