> 精心策划的语法树列表, **[unist][]**, **[mdast][]**
> (markdown), **[hast][]** (HTML), **[xast][]** (XML), **[esast][]** (JS), and
> **[nlcst][]** (prose) resources.

**syntax-tree** 是一个组织住房 **[unist][]** 及其主要实现 **[mdast][]**, **[hast][]**, **[xast][]**, **[esast][]**, 和
**[nlcst][]**.
**[unist][]** 是语法树的规范，它附带了一个大型的 JavaScript 实用程序生态系统，用于处理这些树。
在 **[unist][]** 及其实现之上，还有 **[unified][]** 的其余部分，它们处理各种内容。

## 官方

- [syntax-tree](https://github.com/syntax-tree) - 组织.
- [unist](https://github.com/syntax-tree/unist) - **unist** 规格.

- [xast](https://github.com/syntax-tree/xast) - **xast** (XML) 规格.
