## MDX 做什么？

<div className="big-columns">
  <div>
    您可以通过JSX编写带有嵌入式组件的Markdown：

    ```mdx path="example.mdx"
    import {Chart} from './snowfall.js'
    export const year = 2023

    # 去年的降雪

    在{year}，降雪量高于平均水平。
    随后是一个温暖的春天，导致附近许多河流出现洪水。

    <Chart year={year} color="#fcb32c" />
    ```

  </div>

  <div>

## 简而言之，MDX

<div className="emoji-list">
  * ❤️ **强大的**: MDX融合了markdown和JSX语法，完美地适用于基于JSX的项目
  * 💻 **一切都是一个组件**: 使用MDX中的现有组件，并将其他MDX文件作为组件导入
  * 🔧 **可定制的**:决定为每个标记构造渲染哪个组件 (`{h1: MyHeading}`)
  * 📚 **基于Markdown**: markdown的简洁和优雅仍然存在，只有当您想使用JSX时
  * 🔥 **燃烧得飞快**: MDX没有运行时，所有编译都发生在构建阶段
</div>
