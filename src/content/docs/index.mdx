这些文档解释了 MDX 的核心概念。格式的工作原理，如何将其添加到您的网站，如何使用 MDX 文件，以及如何扩展它们。通读这些应该会让你对 MDX 有一个很好的理解。

1. [什么是 MDX？](./docs/what-is-mdx)

   本文解释了什么是 MDX 格式。它展示了如何在 MDX 中使用 ESM 中的 markdown、JSX、JavaScript 表达式以及导入和导出语句。

2. 开始

   本文介绍了如何将 MDX 集成到您的项目中。它展示了如何将 MDX 与您选择的 bundler 和 JSX 运行时一起使用。

3. 使用 MDX

   本文介绍了如何在项目中使用 MDX 文件。它显示了如何传递属性以及如何导入、定义或传递组件。

4. 扩展 MDX

   本文解释了如何扩展 MDX 内容，特别是如何使用插件转换内容。

5. MDX 故障排除

   本文介绍了使用 MDX 时可能出现的几个常见问题和错误。
